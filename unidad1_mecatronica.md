# Presentación general del curso

Los _sistemas mecatrónicos_ cuentan con múltiples aplicaciones, desde la
automatización de procesos de producción en fábricas, hasta el uso de tecnologías
agrícolas avanzadas para aumentar la productividad en el campo, la minería y el
sector energético.

Asimismo, su implementación puede contribuir a la reducción de costos, al aumento
de la seguridad y a la optimización de los recursos, lo cual permite mejorar la
eficiencia y la competitividad en los diferentes sectores económicos e industriales
del país.

En este sentido, a lo largo del curso realizaremos un proyecto integrador, aplicando
los principios básicos de la mecánica, la electrónica y la informática. Para ello, vamos
a utilizar diferentes herramientas tecnológicas, como dispositivos y _software_ de
diseño y simulación, con el fin de evaluar el rendimiento de los sistemas.

El objetivo principal de esta unidad es resolver problemas de forma rápida y creativa,
trabajar en equipo y liderar proyectos multidisciplinarios. Estas competencias son
fundamentales para hacer frente a los desafíos de diseño, aplicación y
mantenimiento de sistemas mecatrónicos en el ámbito académico y profesional.

# Objetivos de Aprendizaje

Comprender los conceptos básicos de los sistemas mecatrónicos, así como su diseño,
integración y programación, para aplicarlos de manera efectiva en proyectos de
mecatrónica.

## Resultado de aprendizaje

Desarrolla competencias para aplicar principios de control automático en sistemas
mecatrónicos, optimizando procesos autónomos con técnicas y métodos de diseño
modernos.

# Introducción a la unidad

Los sistemas incorporan microprocesadores para su control y sensores eléctricos que
obtienen información sobre las entradas y salidas mecánicas. Esta información se
dirige a los sistemas mecánicos a través de actuadores.

Actualmente, esta integración es usada en el diseño de autos, robots, máquinas,
herramientas, electrodomésticos y muchos otros dispositivos, dando un nuevo
enfoque al diseño de ingeniería, y reemplazando en cierta medida al proceso
tradicional y secuencial de diseño donde los sistemas mecánicos, eléctricos y de
control se hacen por partes.
En la presente unidad conocerás la forma en que estos sistemas se integran en la
mecatrónica, con la finalidad de disminuir sus costos y aumentar la confiabilidad y
flexibilidad.

En la mecatrónica se reúnen diversas tecnologías: la ingeniería mecánica,
electrónica, eléctrica, en computación y de control para diseñar y construir sistemas
automatizados que combinen componentes mecánicos y electrónicos.

La mecatrónica ofrece una alternativa de ver los problemas ingenieriles desde una
perspectiva diferente, donde los ingenieros no se limiten a considerar el desarrollo
de una idea solo desde principios mecánicos, sino también en función de la
electrónica, el control y su respectiva programación. Desde la fase del diseño es
necesario adoptar un enfoque mecatrónico, repensar por completo las necesidades
en términos de lo que se espera de cada componente.

Los principios fundamentales de la mecatrónica incluyen:

**1. Combinación de disciplinas:** interdisciplinariedad entre la ingeniería
    mecánica, electrónica y computación para desarrollar sistemas
    automatizados.
**2. Diseño sistemático:** identificación de requisitos, especificaciones técnicas de
    componentes mecánicos y electrónicos, implementación y puesta a punto del
    sistema.
**3. Sistemas interactivos:** interacción con el entorno mediante el uso de
    actuadores y sensores, así como sistemas de control que responden a
    entradas y realizan acciones específicas.
**4. Optimización del sistema:** incluye aspectos de ahorro de energía, velocidad,
    precisión y fiabilidad.
**5. Flexibilidad y adaptabilidad:** adaptación a diferentes condiciones y
    requisitos, ampliando el horizonte del sistema a otros escenarios y contextos.
**6. Interfaz hombre - máquina:** desarrollo de interfaces intuitivas hombre -
    máquina para facilitar el manejo del usuario final.

## Sistemas de medición

Los sistemas de medición están conformados por tres elementos:

```
● Un sensor que responde a la cantidad medida, generando una señal
relacionada con dicho valor.
● Un acondicionador de señal que toma la señal del sensor y la convierte a una
forma adecuada para su presentación visual y acción de control.
● Un sistema de representación visual o pantalla, donde se muestra la salida
producida por el acondicionador de señal.
```

## Sistemas de control


Un ejemplo muy claro de un sistema de control es el cuerpo humano, donde la
temperatura en su interior debe permanecer constante. Para mantener ese valor, el
cuerpo cuenta con un sistema de control, si la temperatura sube del valor normal,
suda; si disminuye, genera escalofríos. Estos dos mecanismos sirven para restaurar
la temperatura a su valor normal. Este sistema recibe una entrada enviada por
sensores que determinan un valor, y compara estos datos con el valor que debe
tener. A continuación, el sistema produce la respuesta adecuada a fin de lograr la
temperatura requerida.

## Sistemas en lazo cerrado y en lazo abierto

Existen dos tipos básicos de sistema de control: en lazo cerrado y en lazo abierto.

Un _sistema de lazo cerrado_ es aquel en que la salida del sistema es medida y
comparada con una referencia deseada, la cual genera un error y se utiliza para el
ajuste y control de un sistema en tiempo real.


Lo anterior permite una mayor precisión y capacidad de respuesta del sistema ante
cambios en condiciones de operación. Este sistema de control es muy utilizado en
sistemas de control de velocidad para motores eléctricos y sistemas de
posicionamiento para robots industriales.

El _sistema de lazo abierto_ en su salida no contrasta referencias de valores, lo cual lo
vuelve un sistema poco preciso y es usado en aplicaciones donde las condiciones son
constantes y predecibles.

La configuración de sistemas mecatrónicos es una etapa fundamental que implica
orden, selección y combinación de componentes mecánicos, electrónicos y de
_software_ (control), para el óptimo diseño de sistemas integrados, flexibles, precisos
y confiables.

Este tipo de sistemas integrados se encuentran en varias ramas de la industria como
la manufactura, robótica, automatización, biomedicina, entre otras.

El enfoque de carácter interdisciplinario juega un papel crucial en la mejora de
producción, calidad, seguridad, y tiempos en diversos procesos productivos.

En este apartado explicaremos los componentes fundamentales de un sistema
mecatrónico desde su parte mecánica, electrónica y de software, bajo el enfoque de

## Software CAD/CAE /CAM

Un software es una herramienta que asiste el proceso de diseño, análisis y
fabricación de componentes y sistemas mecánicos. Realmente, por sus siglas
muestra las etapas que se viven en el proceso de desarrollo de productos:


CAD (Diseño asistido por computadora): esta herramienta es utilizada para crear
modelos tridimensionales de componentes mecánicos, ensambles y sistemas
completos. Los ingenieros mecatrónicos se valen de este tipo de herramientas para
generar componentes con alto grado de precisión. Luego, integrar partes, y
finalmente, desarrollar un proyecto completo con formas óptimas, ajustes precisos
y con una visión general del proceso, antes de pasar a la fase de fabricación física.

CAE (Análisis por elementos finitos): este tipo de _software_ permite realizar
simulación muy acertadas sobre análisis de tensión, vibración, flujo de claro,
comportamiento estático y dinámico. Estos análisis le permiten al ingeniero evaluar
el rendimiento, resistencia de los componentes a condiciones de carga establecidas,
lo que ayuda a mejorar el proceso de optimización del producto.

CAM (Fabricación asistida por computadora): a partir de modelos 3D y con este tipo
de software simula procesos de fabricación con máquinas y herramientas como
torno y fresadoras, permitiendo optimizar el proceso de fabricación al minimizar
desperdicio de material, disminuir tiempos y generar un proceso de fabricación más
rentable y de mayor calidad.

Las ventajas de integrar este tipo de _software_ a la configuración de sistemas
mecatrónicos incluyen varios aspectos:

```
● Realizar múltiples iteraciones en el diseño de forma rápida y eficiente.
```
```
● Simulación y validación del comportamiento del sistema bajo diferentes
cargas y condiciones de trabajo.
```
```
● Optimización del proceso de fabricación mediante la programación de
máquinas y herramientas.
```
```
● Fomentar el trabajo colaborativo entre equipos multidisciplinarios que
buscan mejorar el ciclo de vida del producto.
```
Los _softwares_ CAD, CAE, CAM cumplen un papel crucial en el desarrollo y diseño de
sistemas mecatrónicos, facilitando el análisis, fabricación de componentes y
sistemas completos, lo que permite a los ingenieros desarrollar productos
innovadores, flexibles y con alto grado de confiabilidad y eficiencia.

## Impresión 3D, prototipado rápido y técnicas de fabricación

## digital

El desarrollo de nuevas tecnologías de cuarta revolución industrial, como la
impresión 3D, el prototipado rápido y las técnicas de fabricación digital, han
transformado de forma radical la configuración de sistemas mecatrónicos,
ofreciendo una gama de beneficios sin precedentes en el desarrollo de productos.

La impresión 3D, también conocida como fabricación aditiva, permite la creación de
piezas tridimensionales de forma rápida y precisa a partir de modelos digitales. Esta
tecnología ha revolucionado el proceso de diseño y fabricación de sistemas
mecatrónicos, al proporcionar la generación de prototipos funcionales a escala en
cuestión de horas, lo que acelera de manera significativa el ciclo y desarrollo de
productos.

La estrecha relación entre el prototipado rápido y la impresión 3D aumenta la
iteración rápida y la validación temprana en conceptos y desarrollo de productos.
Los ingenieros tienen la posibilidad de generar múltiples diseños en poco tiempo,
facilitando la identificación de problemas potenciales y la optimización del
rendimiento para producción a gran escala.


Al igual que la impresión 3D, el mecanizado CNC (Control numérico por
computadora) y el corte láser, pertenecen a las técnicas de fabricación digital. Estas
técnicas permiten la fabricación de piezas finales con una gran variedad de
materiales, que pueden integrarse directamente en sistemas mecatrónicos,
ofreciendo una mayor flexibilidad en el diseño y una calidad superior en la
fabricación.

En conclusión, la impresión 3D, el prototipado rápido y las técnicas de fabricación
digital, han democratizado el proceso de diseños y fabricación en sistemas
mecatrónicos. La generación de comunidades de aprendizaje alrededor de estos
temas avanza a niveles sorprendentes, generando una filosofía de “construir
transbordadores espaciales en tu cocina”, idea que abre nuevas fronteras en la
creación de productos inteligentes y conectados.

## Componentes electrónicos


Los componentes electrónicos, en un sistema mecatrónico, son los encargados de
controlar el funcionamiento e interacción de los componentes mecánicos. Se

## Conceptos básicos de configuración

**Identificación de componentes**

En la configuración de un sistema mecatrónico, el proceso de identificar
componentes es fundamental en el aseguramiento de calidad y funcionamiento.
Este proceso implica una selección cuidadosa desde los componentes mecánicos,
hasta los dispositivos electrónicos y de control, lo anterior enmarcado en un
contexto que garantice un trabajo armónico, y cumplir con los objetivos del
sistema. Los pasos para seleccionar componentes son:


**1.** Definición de requisitos y especificaciones: se debe identificar las funciones
    que el sistema debe realizar, el contexto en los que operará y las
    características del rendimiento y calidad que debe cumplir.
**2.** Análisis funcional del sistema: en esta etapa se divide el funcionamiento del
    sistema en pequeñas tareas, lo que permite comprender cómo interactúan
    los diferentes componentes, y su importancia en el funcionamiento general.
**3.** Selección de componentes mecánicos: se seleccionan estructuras,
    transmisiones, actuadores y mecanismos que generan movimiento. Lo
    anterior considerando factores de resistencia, rigidez, precisión y
    compatibilidad entre componentes que interactúan en el sistema.
**4.** Elección de dispositivos electrónicos y sensores: se eligen los tipos de
    sensores que pueden ser de posición, velocidad, fuerza, temperatura, entre
    otros, así como motores, actuadores. Se debe tener en cuenta la precisión,
    robustez y respuesta dinámica del sistema.
**5.** Diseño del sistema de control: se seleccionan los algoritmos de control,
    controladores y actuadores necesarios, para el óptimo funcionamiento del
    sistema mecatrónico.
**6.** Integración y pruebas: etapa final donde se integran todos los componentes
    mecánicos, electrónicos y control del conjunto. Se realizan pruebas
    exhaustivas, para garantizar el óptimo funcionamiento del sistema.

## Conexiones y comunicaciones entre dispositivos

A través de diferentes protocolos de comunicación y tecnologías de transferencias
de datos y señales, los componentes mecánicos, electrónicos y de software,
interactúan de forma sincrónica en el funcionamiento de un sistema mecatrónico.
Algunas de las formas más comunes de conexión y comunicación entre dispositivos
en sistemas mecatrónicos son:

**1. Buses de campo:** son sistemas de comunicación para entornos industriales y
    mecatrónicos, que enlazan dispositivos de entrada como sensores,
    actuadores, etc, con dispositivos de salida como válvulas, lámparas
    indicadoras, accionamientos entre otros, sin necesidad de conectar cada
    dispositivo a un controlador central. Estos buses permiten la comunicación
    entre dispositivos en tiempo real. En la _Figura 13_ se observa un esquema de
    un bus de campo.

**2.** Ethernet: se utiliza para comunicaciones mecatrónicas más complejas, esta
    ofrece una mayor capacidad y velocidad de transferencia de datos.
    Recomendado para sistemas que requieren una gran cantidad de información
    que debe transmitirse de manera rápida y confiable.
**3.** Inalámbrico: la comunicación inalámbrica puede utilizarse cuando el sistema
    tiene limitaciones de espacio o de diseño para conectar dispositivos sin
    necesidad de cables. Entre las tecnologías que hacen parte de este tipo de
    comunicación tenemos _Wi - Fi, Bluetooth, Zigbee_ y radio frecuencia.
**4.** Redes de control distribuido: utilizadas en sistemas mecatrónicos grandes y
    complejos, se divide el sistema en módulos que son interconectados por una
    red de comunicación, permitiendo mayor flexibilidad en el diseño y control
    del sistema.

**Configuración de** **_hardware_** **y** **_software_** **(Interactivo)**

El funcionamiento coordinado y eficiente de todos los componentes de un sistema
mecatrónico, depende en gran medida de la configuración y alistamiento de
_hardware_ y _software_. A continuación, se muestra una descripción general de cómo
se realiza esta configuración:

## 1. Hardware

```
● Selección de componentes: es importante llevar a cabo un análisis detallado
para seleccionar los componentes mecánicos, electrónicos y eléctricos que
conformarán el sistema mecatrónico. Para realizar una elección idónea, es
necesario tener en cuenta los requisitos de diseño, el presupuesto disponible
y el rendimiento deseado, verificando que cada elemento cumpla con las
especificaciones técnicas.
```
```
● Integración de los componentes: este paso implica la instalación física de los
elementos clave del sistema, tales como sensores, actuadores, controladores
y unidades de procesamiento. La integración de todos los componentes
garantiza que el sistema funcione de manera coherente y competente.
```
```
● Interconexión y cableado: a continuación, se define y establece la
interconexión entre los diferentes componentes, incluyendo las conexiones
eléctricas, mecánicas y de comunicación. En este proceso es esencial prestar
atención a la elección y el uso adecuado de conectores, buses de campo,
cables y otras tecnologías de conexión, de modo que se logre asegurar una
transmisión eficiente y segura de los datos y la energía.
```
## 2. Software

```
● Programación de controladores: aquí, se desarrollan los algoritmos de
control necesarios para gestionar los dispositivos de control del sistema, tales
como controladores lógicos programables (PLC), microcontroladores y
matrices de campo programables (FPGA). La programación cuidadosa permite
```

```
mantener el funcionamiento correcto del sistema, así como la automatización
y el control preciso de los procesos.
```
```
● Desarrollo de la interfaz de usuario: se diseña y prepara el software para la
interacción del usuario con el sistema mecatrónico. Las interfaces pueden
contener pantallas táctiles, paneles de control, así como aplicaciones de
escritorio y móviles, de acuerdo con las necesidades del cliente y las
características del sistema.
```
```
● Sincronización de sistemas: se desarrollan e implementan los protocolos de
comunicación que permitirán la integración efectiva de los componentes del
sistema mediante software. Para este proceso se requiere la configuración de
redes de difusión y la creación de interfaces de aplicación (API) que garanticen
una interoperabilidad fluida entre todos los elementos del sistema.
```
```
● Pruebas y depuración: se efectúan pruebas exhaustivas en diferentes niveles:
unitarias, de integración y del sistema completo. El objetivo es detectar y
corregir cualquier error o problema de rendimiento para que el sistema
funcione según lo esperado antes de su implementación final.
```
